# Kingdom Clicker (alpha)

Lightweight, headless-friendly kingdom builder with an optional Tk UI. Run `python -m build` to package or `python kclicker.py` to play.

## Install latest release via pipx

Use the helper script to grab the newest GitHub release wheel:

```
python scripts/install_latest.py
```

To only print the resolved wheel URL (no install), run:

```
python scripts/install_latest.py --print-only
```
